🌐 LinguaVoice AI – Language Translator Web Application

📌 Project Overview

LinguaVoice AI is a modern and interactive Language Translator Web Application developed using HTML, CSS, and JavaScript. The project is designed to help users translate text between multiple languages through a clean, responsive, and user-friendly interface. The application combines attractive UI design with useful translation features, making communication across different languages easier and more accessible.

This project was developed as part of the CodeAlpha Internship Program, providing hands-on experience in frontend web development and real-world project implementation. Through this project, I gained valuable experience in designing responsive user interfaces, implementing interactive features, and improving user experience using modern web technologies.

---

🚀 Features

🌍 Multi-Language Translation

- Translate text between multiple languages.
- Simple language selection using dropdown menus.
- User-friendly translation workflow.

🎤 Voice Input Support

- Built-in speech recognition functionality.
- Users can speak instead of typing.
- Converts spoken words into text automatically.

🔊 Text-to-Speech Output

- Listen to translated text using browser speech synthesis.
- Supports natural voice playback.
- Enhances accessibility and usability.

🔄 Language Swap Feature

- Instantly swap source and target languages.
- Improves translation convenience.

📋 Copy Translation

- Copy translated text to the clipboard with a single click.
- Easy sharing and reuse of translated content.

📱 Responsive Design

- Fully responsive layout.
- Works smoothly on desktops, tablets, and mobile devices.
- Optimized user experience across different screen sizes.

🎨 Modern User Interface

- Beautiful gradient effects and animations.
- Glassmorphism-inspired design.
- Dark mode and light mode support.
- Smooth transitions and interactive elements.
